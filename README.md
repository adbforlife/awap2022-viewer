# AWAP 2022 Viewer

Viewer for AWAP 2022 replay files

## How-to:
1. Clone this repo
2. `cd` into this repo
3. Run `npm install`
4. Run `npm start`
5. Navigate to http://localhost:8080/
6. For fabric.js version, navigate to http://localhost:8080/test
